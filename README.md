# 🚀 CryptoDash Ultimate - ENSAH
Application de suivi de crypto-monnaies en temps réel développée dans le cadre de ma formation d'ingénieur à l'**ENSAH**.

## ✨ Fonctionnalités
- **Données en direct** : Récupération des prix via l'API CoinGecko.
- **Conversion MAD** : Support complet pour le Dirham Marocain.
- **Graphiques interactifs** : Visualisation avec Chart.js.
- **Mode Sombre/Clair** : Interface adaptative.

## 🛠️ Technologies
- HTML5 / CSS3 (Design moderne)
- JavaScript (Asynchrone / Fetch API)
- Git & GitHub Pages (Déploiement continu)
